# 一个开箱即用的H5项目模板
使用uniapp+uview1+vue2

## 快速开始
```
安装
npm install
cnpm i (可能会出现包版本不兼容问题)
```

## 运行
```
小程序
npm run dev:mp-weixin

h5
npm run dev:h5
```

## 打包
```
小程序
npm run dev:mp-weixin

h5
npm run dev:h5
```

## 部署
```
小程序
npm run deploy:mp-weixin
```

## 版本
```
node v14.19.3

npm v6.14.17
```
